Customer Shopping Behavior Analysis: Strategic Insights & Recommendations

📌 Project OverviewThis project performs a deep-dive analysis into customer shopping behavior using a dataset of 3,900 purchases. The objective is to uncover spending patterns, demographic influences, and product performance to drive strategic business growth.

🛠️ Technical StackProgramming: Python (Pandas, SQLAlchemy)   Database: PostgreSQL / MySQL for structured querying   Visualization: Power BI for interactive dashboarding   Environment: Jupyter Notebook / Terminal

📊 Key Insights & Findings

Demographic Drivers: Male customers contributed significantly more revenue ($157,890) than female customers ($75,191).

Age Group Impact: "Young Adults" are the leading revenue contributors ($62,143), followed closely by "Middle-aged" shoppers.

Product Performance: Top-rated items include Gloves (3.86), Sandals (3.84), and Boots (3.82).

Subscription Status: 73% of customers are non-subscribers, presenting a massive opportunity for conversion strategies.

Logistics: Express shipping users show a higher average purchase amount ($60.48) compared to standard shipping users ($58.46).

📂 Repository Structure

portfolio.py: Python script for data cleaning, feature engineering (age binning, frequency mapping), and database loading.

Customer Shopping Behavior Analysis.pdf: Comprehensive project report containing detailed SQL findings and Power BI visuals.

customer_shopping_behaviour.csv: The raw dataset containing 18 key features.

🚀 How to Run the Analysis

1. Python Processing: Run portfolio.py to clean the data and handle missing review ratings via category-specific medians.

2. Database Integration: Configure your create_engine parameters in the script to load data into your local database.

3. Visuals: Open the Power BI dashboard to explore interactive revenue and customer segmentation visuals.

💡 Business Recommendations

Retention: Develop loyalty programs to convert the 701 "Returning" customers into the "Loyal" segment.

Subscription Growth: Promote exclusive benefits to the 2,847 non-subscribers to increase recurring revenue.

Targeted Marketing: Focus marketing efforts on high-revenue age groups (Young Adults) and high-value shipping users.
